# 👯‍♀️Stack Overflow Clone Project

[스택오버플로우](https://stackoverflow.com/questions)의 일부를 클론 코딩합니다.

<br/><br/>

## 🥅 Our Goal

- 질문을 생성하고 삭제할 수 있습니다.
- 질문에 답변을 생성하고 삭제할 수 있습니다.
- create-react-app으로 간편한 번들링과 배포가 가능합니다.
- Spring Boot로 쉽게 서버 배포를 할 수 있습니다.
